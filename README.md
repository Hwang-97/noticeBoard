# noticeBoard
## 해당 레파지토리는 수업중 진행한 코드로 게시판 사이트의 기초적이면서 전반적인 기능을 구현한것 입니다.
___
사용한 기술로는 
1. jsp
2. css
3. jquery
4. oracle
<br>
<hr>
<br>기초적인 CRUD를 구현하였으며 게시물 마다 조회수, 검색 등 기초가 되는 모든 기능을 구현되어 있습니다.
